Everything you need

Built for modern learning
operations

A complete toolkit for admins, instructors, and learners.

Role-based access control
Separate dashboards for super admins, category admins, and learners. Every user sees exactly what they need.
Course and content management
Create, organise, and publish courses with module-level control. Manage tiers, statuses, and learner access in real time.
Analytics and progress tracking
Real-time PAL scores, quiz averages, completion rates, and leaderboards. Identify at-risk learners before they fall behind.
Moodle and API integration
Sync categories, courses, and users directly with Moodle. Open API support for third-party integrations.
Enrollment management
Manual and self-enrollment flows with approval queues, domain-based access control, and built-in CSV verification.
