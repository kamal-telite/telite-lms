Telite Systems — Telite Admin
Core features (refined):

Category management — create, edit, archive, and delete categories like ATS
Dashboard stats — total courses, active categories, and system-wide enrollment count
User overview — number of learners per category, with drill-down
Admin control — appoint or revoke admin roles for any category

Production additions:

Role-based access control (RBAC) so super admin permissions are clearly separated from category admins
Soft-delete on categories (archive before permanent removal) to prevent accidental data loss
Activity log / audit trail for all admin actions
Global PAL aggregate — see performance summaries across all categories from one view
